## Installation

* from pypi: `pip install python-mkblog`
* from github: `pip install git+https://github.com./derJD/python-mkblog.git`

### Local Build

```sh
git clone https://github.com/derJD/python-mkblog.git
cd python-mkblog
pip install .
```
